# 枪手（Gunner）职业实现分析

## 核心类：CNGunner（line 368706）

```csharp
public sealed class CNGunner : IRDCharacter
```

继承自 `IRDCharacter`，是一个 **sealed** 类（不可再被继承），通过 `ENUM_GROW_TYPE` 枚举区分转职：

| 转职 | 枚举值 | AI 控制器 |
|------|--------|----------|
| 基础枪手 | `GROW_TYPE_GUNNER` | `IRDGunnerAIController` |
| 漫游枪手 (Ranger) | `GROW_TYPE_RANGER` | `IRDRangerAIController` |
| 弹药专家 (Spitfire) | `GROW_TYPE_SPIT_FIRE` | — |
| 机械师 (Mechanic) | `GROW_TYPE_MECHANIC` | — |
| 枪炮师 (Launcher) | `GROW_TYPE_LAUNCHER` | `IRDLauncherAIController` |

## 武器系统

### 主武器类型（ENUM_WEAPON_SUBTYPE）

- `REVOLVER` — 左轮
- `AUTOMATIC` — 自动手枪
- `MUSKET` — 步枪（慢速高伤）
- `HAND_CANNON` — 手炮
- `BOW_GUN` — 弓弩

### 副武器类型（ENUM_SUB_WEAPON_TYPE）

10 种，包括 `FLAME`、`GATLING`、`LASER`、`STEYR`、`STINGER` 等。

### GunInfo 结构（line 368670）

管理枪械参数：

- `shootDelays_[]` — 每发射击间隔
- `drawSpeed_` / `reloadSpeed_` — 拔枪/换弹速度
- `pierceForceOfGunnerGun_` — 子弹穿透力（决定可穿透几个敌人）
- `attackMaxNumber_` — 最大连击数
- `jumpAttackRecoils_[]` — 跳射后坐力

## 射击与攻击系统

### 射击方向（ENUM_GUN_SHOOT_TYPE）

- `UP` / `DOWN` / `JUMP` / `TREAD`

每种方向有独立的距离配置数组：

```
ATTACK_UP_GUN_DISTANCE[]        // 上射枪距
ATTACK_DOWN_BULLET_DISTANCE[]   // 下射弹距
JUMP_ATTACK_MUZZLE_DISTANCE[,]  // 跳射枪口距
```

### 子弹类

- `CNGunnerBullet`（line 446815）— 主武器子弹，有穿透、旋转角度、银弹特效等属性
- `CNGunnerSubWeaponBullet`（line 442047）— 副武器子弹，按副武器类型有不同行为（火焰喷射器动画、FM31手雷伤害加成等）

### 连击系统

`attackNumber_` 跟踪当前连击段数，`attackMaxNumber_` 限制上限，地面和空中连击独立计算。

## 特殊机制

### 附魔系统（Appendage）

- `BulletBuffAppendage` — 子弹附魔
- `SiegeModeAappendage` — 围城模式（固定炮台）
- `revolverCriticalDamageUpAppendage_` — 左轮暴击加成
- `stylishCriticalUpAppendage_` — 华丽暴击提升

### 防御机制

- `isInBulletGuard_` — 弹幕防御状态
- `currentBulletGuardDamageRate_` — 防御时伤害减免率

### AI 行为（line 363145+）

- `IRDGunnerAIController` 通过 `AISkillDivideCheckerA/B/C` 进行技能选择分级
- `FSMUnitCharacterGunnerIsNextShoot` FSM 状态判定是否应该继续射击
- 各转职有独立 AI 控制器覆写技能策略

## 总体架构

```
IRDCharacter (基类)
  └── CNGunner (枪手)
        ├── GunInfo (枪械配置)
        ├── CNGunnerBullet / CNGunnerSubWeaponBullet (子弹)
        ├── 技能类 (Skill → ShootSkill / UseSubWeaponSkill / PassiveSkill / StackableSkill)
        ├── Appendage 系统 (Buff/状态附魔)
        ├── IRDGunnerAIController (AI 控制)
        └── ENUM_GROW_TYPE (转职分支)
```

枪手通过 **同一个 CNGunner 类 + 转职枚举** 实现所有分支，而不是为每个转职创建子类。技能、AI、副武器的差异化行为通过转职类型判断和不同的 AI 控制器来实现。
