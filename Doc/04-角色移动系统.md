## 3. 每帧移动更新流水线

### 核心更新方法

```
updateMoveObject(moveDeltaTime)
  ├── updateMoveValues(moveDeltaTime)   ← 速度/加速度计算
  └── updateCurrentPos(moveDeltaTime)   ← 位置积分 + 碰撞校验
```

### 详细流程

```
1. 输入/AI 确定目标方向
       ↓
2. 状态机判断当前状态是否允许移动 (isMovableState())
       ↓
3. getMoveSpeed() 计算实际速度（含 Buff 加减速）
       ↓
4. updateMoveValues():
   ├── 根据 MOVESTATE 应用加速/减速
   ├── ACCEL_UP:   velocity += accel × dt（加速到匀速）
   ├── UNIFORM:    velocity = 常量（匀速运动）
   ├── ACCEL_DOWN: velocity -= accel × dt（减速到停止）
   └── 计算新速度向量
       ↓
5. updateCurrentPos():
   ├── 新位置 = 旧位置 + velocity × dt
   ├── checkMovablePos() 碰撞合法性检查
   ├── 应用重力/Z轴加速度（如果在空中）
   ├── 限制在地图边界内
   └── 更新 lastXPoses_ 位置历史
       ↓
6. 方向变更处理 (isDirectionChangeableState())
       ↓
7. 网络同步发送 MovementInfo（多人模式）
       ↓
8. 动画更新（根据速度+方向）
```

## 4. 速度系统（IRDActiveObject）

### 速度类型（ENUM_SPEED_TYPE）

```csharp
SPEED_TYPE_MOVE_SPEED                   // 移动速度
SPEED_TYPE_ATTACK_SPEED                 // 攻击速度
SPEED_TYPE_CAST_SPEED                   // 施法速度
SPEED_TYPE_EXCEPT_WEAPON_ATTACK_SPEED   // 非武器攻击速度
SPEED_TYPE_CONST                        // 固定速度（不受加成影响）
```

### 速度字段

```csharp
private ENUM_SPEED_TYPE moveSpeedType_;       // 当前速度类型
public Int32 moveSpeedValue_;                 // 基础速度值
private Single moveSpeedBonusRate_;           // 速度加成倍率（Buff/Debuff）
public Int32 minmumMoveSpeed_;                // 最低速度下限

// 动画速度（独立于移动速度）
public Int32 animationSpeedValue_;
public Single animationSpeedBonusRate_;
public Int32 minmumAnimationSpeed_;
```

### 速度计算

```
实际速度 = max(moveSpeedValue_ × (1.0 + moveSpeedBonusRate_), minmumMoveSpeed_)
```

速度类型为 `SPEED_TYPE_CONST` 时不受 `moveSpeedBonusRate_` 影响，用于固定速度的移动（如击退位移）。

```csharp

```

### 跳跃状态流程

```
STATE_JUMP_READY  → 预备（蓄力/动画）
    ↓
STATE_JUMP_UP     → 上升（Z速度 > 0，重力减速）
    ↓
                    到达最高点（Z速度 = 0）
    ↓
STATE_JUMP_DOWN   → 下降（重力加速）
    ↓
STATE_JUMP_STAND  → 落地（Z位置 = 0）
    ↓
STATE_BREAK_FALL  → 落地恢复动画
```

## 7. 方向系统

### 方向枚举

`ENUM_DIRECTION_MOBILE` 支持 8 方向或 16 方向移动。

### 方向相关方法

```csharp
SetDirection(ENUM_DIRECTION_MOBILE direction)     // 设置朝向
GetDirection()                                     // 获取当前朝向
getInputDirection(Int32 axis)                      // 获取输入方向（按轴）
getHitDirection(IRDCollisionObject damager)        // 获取受击方向
isDirectionChangeableState()                       // 当前状态是否允许转向
```

### 方向与移动的关系

- 移动方向和面朝方向可以独立（如走射时面朝敌人但身体侧移）
- 部分状态下锁定方向（如技能释放中不可转向）
- 受击时自动调整面朝方向（面向攻击者）

## 8. A* 寻路移动（AStarMoveController）

主要用于 **AI 和 NPC 的自动移动**：

```csharp
public class AStarMoveController (line 308232)
{
    private IRDActiveObject controlTarget_;        // 控制目标
    private Queue<Vector3> astarPathCheckPoint;    // 路径点队列
    private Vector3 startPos_;                     // 起点
    private Vector3 targetPos_;                    // 终点
    private ENUM_DIRECTION_MOBILE oldDirection_;   // 上一方向
    private Boolean isDash;                        // 冲刺标记
}
```

### 关键方法

```csharp
TryPathFind(IRDActiveObject target, Vector3 targetDNFPos)  // 尝试寻路
StartMove(Vector3 targetPos, Int16 uniqueID)               // 开始移动
updateAutomaticMove(Single logicDeltaTime)                 // 每帧自动移动
getNextMoveDirection(Vector3 current, Vector3 dest, ENUM_DIRECTION_MOBILE destDir) // 计算下一方向
OnPathComplete(Vector3 end, List<Vector3> path)            // 寻路完成回调
```

### 寻路流程

```
TryPathFind() → A*计算路径
    ↓
OnPathComplete() → 路径点入队 astarPathCheckPoint
    ↓
每帧 updateAutomaticMove():
  ├── 取队首路径点
  ├── getNextMoveDirection() 计算移动方向
  ├── 向路径点移动
  ├── 到达后出队，取下一个
  └── 全部到达后停止
```

### 击退处理

```csharp
private Vector2 knockBackLimitPos;  // 击退边界限制
```

击退使用 `MOVESTATE_COLLISION` 状态，速度类型切换为 `SPEED_TYPE_CONST`（不受速度加成影响），按固定速度和时间进行位移，到达 `knockBackLimitPos` 边界或时间结束后恢复正常状态。

## 12. 网络同步

### MovementInfo 结构

```csharp
public class MovementInfo
{
    public Byte flag;              // 移动更新标记
    public Int16 groupid;          // 对象组ID
    public Int16 uniqueid;         // 对象唯一ID
    public Byte direction;         // 当前方向
    public Byte moveState;         // 当前 MOVESTATE
    public Vector3 pos_;           // 位置 (X, Y, Z)
    public Vector3 velocities_;    // 速度（每轴）
    public Vector3 accels_;        // 加速度（每轴）
    public Vector3 accelValidTime_; // 加速度有效时长
}
```

同步策略：发送当前位置 + 速度 + 加速度，接收方通过插值预测位置，减少网络延迟感。

## 13. 移动约束汇总

| 约束 | 实现方式 |
|------|---------|
| 地图边界 | `isMovablePos()` + `CheckEdgeCollision()` |
| 地形碰撞 | `ENUM_PASSAGETYPE` 通过性判定 |
| 速度下限 | `minmumMoveSpeed_` 防止减速到 0 以下 |
| 状态锁定 | `isMovableState()` / `isDirectionChangeableState()` |
| 击退边界 | `knockBackLimitPos` 限制击退最远距离 |
| 网格合法性 | `bUseAstarTile` 寻路网格校验 |

## 整体架构图

```
┌──────────────────────────────┐
│         输入 / AI             │
│  玩家操作 或 AStarMoveController │
└──────────────┬───────────────┘
               ↓
┌──────────────────────────────┐
│      状态机判定                │
│  isMovableState()             │
│  当前状态 → 允许的移动类型     │
└──────────────┬───────────────┘
               ↓
┌──────────────────────────────┐
│      速度计算                  │
│  getMoveSpeed()               │
│  base × (1 + bonusRate)       │
│  min(result, minimumSpeed)    │
└──────────────┬───────────────┘
               ↓
┌──────────────────────────────┐
│  updateMoveValues()           │
│  ┌────────────────────────┐  │
│  │ ACCEL_UP → 加速         │  │
│  │ UNIFORM  → 匀速         │  │
│  │ ACCEL_DOWN → 减速       │  │
│  │ COLLISION → 碰撞位移    │  │
│  └────────────────────────┘  │
│  + Z轴重力: moveZ()          │
└──────────────┬───────────────┘
               ↓
┌──────────────────────────────┐
│  updateCurrentPos()           │
│  newPos = pos + vel × dt      │
│  checkMovablePos() 碰撞校验   │
│  isMovablePos() 地图合法性    │
│  onMoveCollision() 碰撞回调   │
└──────────────┬───────────────┘
               ↓
┌──────────────────────────────┐
│  后处理                       │
│  · 方向更新                   │
│  · 动画同步                   │
│  · 网络同步 MovementInfo      │
│  · 位置历史 lastXPoses_       │
└──────────────────────────────┘
```
