```csharp
{
```

**公式**：
```
最终值 = 加值 × (1 + 倍率)
```

#### HP/MP 特殊公式（支持多层叠加）

```csharp
// HP 最大值计算（line 356478）
public Int64 getHpMax()
{
    // hpMaxRate_ 是数组，支持多层倍率叠加
    // 每层独立计算后累乘
    Int64 base = hpMaxAdd_;
    for (int i = 0; i < hpMaxRate_.Length; i++)
    {
        base = (Int64)(base * (1.0f + hpMaxRate_[i]));
    }
    return base;
}
```

**公式**：
```
HP = Add × (1 + Rate[0]) × (1 + Rate[1]) × (1 + Rate[2]) × ...
```

#### 速度计算（含上下限）

```csharp
// 移动速度计算（line 356504）
public Int32 getMoveSpeed(Boolean useMin)
{
    Int32 base = moveSpeedAdd_;
    Single rate = getMoveSpeedRate();
    
    // 应用倍率
    Int32 result = (Int32)(base * (1.0f + rate));
    
    // 上下限约束
    if (useMin)
    {
        // 有最小速度限制，防止减速到0
        result = Math.Max(result, MIN_MOVE_SPEED);
    }
    
    return result;
}

// 移动速度倍率计算（line 356510）
public Single getMoveSpeedRate()
{
    // 综合多个倍率来源
    Single rate = MoveSpeedRate 
                + equipmentMoveSpeedRate_ 
                + extraSpeedRate_;
    
    // 减速下限保护
    rate = Math.Max(rate, -limitSlowDownMoveSpeedRate_);
    
    // 加速上限约束
    rate = Math.Min(rate, revisionLimitMoveSpeedRate_);
    
    return rate;
}
```

**公式**：
```
速度倍率 = clamp(baseRate + equipmentRate + extraRate, -slowDownLimit, revisionLimit)
最终速度 = max(Add × (1 + 速度倍率), MIN_SPEED)
```

## 七、设计要点总结

### 7.1 Add + Rate 双值设计

**优势**：
- **独立叠加**：多个 Buff 可以独立叠加加值和倍率
- **灵活配置**：同一个 Buff 可以同时提供加值和倍率效果
- **易于撤销**：通过 uniqueId 追踪来源，精确撤销特定 Buff

**计算顺序**：
```
最终值 = (基础值 + Σ加值) × (1 + Σ倍率)
```

### 7.2 对象池与深拷贝

- `ActiveStaticInfo` 和 `STChangeStatusData` 都使用对象池
- 支持深拷贝用于属性快照（PVP 属性锁定、死亡回滚等）
- 减少 GC 压力，支持高频 Buff 创建/销毁

### 7.3 多层倍率叠加

HP/MP 使用数组存储多层倍率，每层独立计算后累乘：
```
HP = Add × (1 + Rate[0]) × (1 + Rate[1]) × (1 + Rate[2])
```

用于区分不同来源的倍率叠加（基础、装备、Buff、技能等）。

### 7.4 速度上下限保护

- `limitSlowDownXxxRate_` — 减速下限，防止被减速到 0
- `revisionLimitXxxRate_` — 加速上限，防止无限加速
- `MIN_SPEED` — 最小速度值兜底

### 7.5 统一查询接口

`GetStat(ENUM_CHAR_STAT)` 提供统一查询入口，支持：
- 类型自动判断（整数/浮点）
- 统一日志和调试
- UI 显示统一格式化

```csharp
{
    // 基础值
    public int BaseHP = 1000;
    public int BaseAttack = 100;
    public int BaseDefense = 50;
    public float BaseMoveSpeed = 3f;
    public float BaseAttackSpeed = 1f;

    public int DefenseAdd;

    public float AttackSpeedAdd;

    public const float MAX_SPEED_RATE = 2f;     // 最大加速 200%

    public int MaxHP => Math.Max(1, (int)((BaseHP + HPAdd) * HPRate));
    public int Attack => Math.Max(0, (int)((BaseAttack + AttackAdd) * AttackRate));
    public int Defense => Math.Max(0, (int)((BaseDefense + DefenseAdd) * DefenseRate));

        }
    }
    
    public float AttackSpeed
    {
        get
        {
            float rate = Math.Clamp(AttackSpeedRate, 1f + MIN_SPEED_RATE, 1f + MAX_SPEED_RATE);
            return Math.Max(0.5f, (BaseAttackSpeed + AttackSpeedAdd) * rate);
        }
    }

    public void Init()
    {
        CurrentHP = MaxHP;
        CurrentMP = MaxMP;
```

```csharp
public class SimpleBuff
{
    public int BuffId;
    public string Name;
    public float Duration = -1f;  // -1 = 永久
    public float RemainingTime;
    
    // 属性变更列表
    public List<StatChange> Changes = new List<StatChange>();
    
    // 特效
    public GameObject StartEffect;
    public GameObject LoopEffect;
    public GameObject EndEffect;
    
    // 应用到属性
    public void Apply(CharacterStats stats)
    {
        foreach (var change in Changes)
        {
            ApplyChange(stats, change, true);
        }
    }
    
    // 从属性撤销
    public void Remove(CharacterStats stats)
    {
        foreach (var change in Changes)
        {
            ApplyChange(stats, change, false);
        }
    }
    
    private void ApplyChange(CharacterStats stats, StatChange change, bool add)
    {
        float value = add ? change.Value : -change.Value;
        
        switch (change.Type)
        {
            case StatType.Attack:
                if (change.ChangeKind == ChangeType.Add)
                    stats.AttackAdd += (int)value;
                else
                    stats.AttackRate *= (1 + value);
                break;
                
            case StatType.MoveSpeed:
                if (change.ChangeKind == ChangeType.Add)
                    stats.MoveSpeedAdd += value;
                else
                    stats.MoveSpeedRate *= (1 + value);
                break;
                
            // ... 其他属性
        }
    }
    
    // 每帧更新
    public void Update(float deltaTime)
    {
        if (Duration > 0)
        {
            RemainingTime -= deltaTime;
            if (RemainingTime <= 0)
            {
                // 标记为待移除
                IsValid = false;
            }
        }
    }
    
    public bool IsValid = true;
}
```

### 10.1 物理攻击力计算流程

```
角色基础物理攻击: 100
装备物理攻击加值: +50
技能 Buff 物理攻击加值: +30
装备物理攻击倍率: +20%
技能 Buff 物理攻击倍率: +15%

计算：
  加值总和 = 100 + 50 + 30 = 180
  倍率总和 = 1 + 0.20 + 0.15 = 1.35
  最终物理攻击 = 180 × 1.35 = 243
```

```csharp
    int baseAttack = attack.isMagical ? attacker.MagicalAttack : attacker.Attack;
    
    // 应用技能倍率
    int damage = (int)(baseAttack * attack.Power);
    
    // 暴击判定
    float critRate = attack.isMagical ? attacker.MagicalCriticalRate : attacker.PhysicalCriticalRate;
    bool isCritical = Random.value < critRate;
    if (isCritical)
    {
        float critDamage = attack.isMagical ? attacker.MagicalCriticalDamageRate : attacker.PhysicalCriticalDamageRate;
        damage = (int)(damage * critDamage);
    }
    
    // 防御减免
    int defense = attack.isMagical ? defender.MagicalDefense : defender.Defense;
    int finalDamage = Math.Max(1, damage - defense);
    
    // 最终伤害倍率
    finalDamage = (int)(finalDamage * attacker.FinalDamageRate);
    
    return finalDamage;
}
```
