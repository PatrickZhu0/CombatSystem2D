### 核心设计要点

- **三级窗口类型** — HUD（非模态）、MainWindow（全屏模态）、Popup（模态弹窗），职责清晰
- **UIElement 统一生命周期** — Init → Open → PostOpen → Close → Release，所有窗口一致
- **UIParam 类型化传参** — 窗口间通过强类型参数传递数据，类型安全
- **伤害数字 Mesh 合批** — 大量数字通过 MeshFilter 一次 Draw Call 渲染，性能优化
- **对象池化** — 伤害数字、特效均使用对象池
- **设备适配** — 安全区域 + 刘海屏 + 预设布局，适配多种移动设备
- **Escape 键层级处理** — 从顶层窗口向下传递，逐层消费
