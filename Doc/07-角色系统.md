## 状态机系统

### 状态枚举：ENUM_ACTIVE_OBJECT_STATE（line 505222）

`Int16` 类型枚举，包含通用状态和职业专属状态。

#### 通用状态

| 状态 | 说明 |
|------|------|
| `STATE_NONE` | 无状态 |
| `STATE_STAND` | 站立 |
| `STATE_REST` | 休息 |
| `STATE_MOVE` | 移动 |
| `STATE_DAMAGE` | 受伤 |

| `STATE_DIE` | 死亡 |
| `STATE_JUMP` | 跳跃 |
| `STATE_JUMP_ATTACK` | 跳跃攻击 |
| `STATE_ATTACK` | 攻击 |
| `STATE_HOLD` | 抓取 |
| `STATE_BUFF` | Buff |
| `STATE_DASH` | 冲刺 |
| `STATE_EVADE` | 闪避 |

## 设计模式总结

| 模式 | 应用 |
|------|------|
| **工厂模式** | `GameObjectLoader` 根据职业/类型创建对应角色实例 |
| **中介者模式** | `ICharacterMediator` 接口解耦角色间交互 |
| **状态机模式** | `ENUM_ACTIVE_OBJECT_STATE` + `addSetState/flushSetState` 驱动行为 |
| **对象池模式** | `PooledLogicRecyclingObject` 基类 + `DNFObjectCreator_Character` 实现复用 |
| **单例模式** | `DNFObjectManager` 等 Manager 类 |
| **委托回调** | 状态变更事件、对象创建/移除事件 |
